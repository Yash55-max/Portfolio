# 🚀 Yashwanth Ponnam | Portfolio

A premium, high-performance developer portfolio built with a focus on system aesthetics, data science, and modern web interactivity.

![Tech Stack](https://img.shields.io/badge/Stack-HTML%20%7C%20CSS%20%7C%20JS-red?style=for-the-badge)

## ✨ key Features

- **Terminal-Inspired Design**: Glassmorphism, grid backgrounds, and custom ASCII art terminal cards.
- **Dynamic GitHub Integration**:
    - Live-fetched pinned repositories with real-time stars and forks.
    - Automated contribution heatmap integration.
    - Profile statistics (followers, total repos, stars) updated on load.
- **Interactive Skills Section**:
    - 50+ localized skill pills with automated logo injection from Devicons and Simple Icons.
    - Animated proficiency bars for core technical domains.
- **Embedded Resume**: Interactive PDF viewer with direct Google Drive integration for viewing and downloading.
- **Custom UI/UX**:
    - Smooth scrolling with magnetic cursor effects.
    - 3D tilt interactions on cards.
    - Responsive mobile-first layout.
- **Contact System**: Fully functional contact form integrated with Formspree for reliable message delivery.

## 🛠️ Tech Stack

- **Core**: HTML5, Vanilla CSS3, Modern JavaScript (ES6+)
- **APIs & Integrations**: 
    - [GitHub REST API](https://docs.github.com/en/rest)
    - [Devicons](https://devicon.dev/) (Dynamic logos)
    - [Simple Icons](https://simpleicons.org/) (Brand logos)
    - [GHChart](https://ghchart.rshah.org/) (Contribution visualization)
- **Deployment**: Optimized for Vercel, Netlify, or GitHub Pages.

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Yash55-max/PORTFOLIO.git
   ```

2. **Customize your information:**
   - Open `index.html` and search for the `PINNED_REPOS` array to list your own repositories.
   - Update your social links and contact details in the `NAV`, `HERO`, and `CONTACT` sections.
   - Replace the Google Drive resume IDs with your own shared file ID.

3. **Run locally:**
   Simply open `index.html` in your browser (no build step required) or use a live server extension.
